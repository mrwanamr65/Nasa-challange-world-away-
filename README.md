# 🌌 A World Away – NASA Exoplanet Classifier

An AI-powered web application developed for the NASA Space Apps Challenge that helps identify and classify exoplanets using machine learning and NASA astronomical datasets.

## 🚀 Project Overview

Thousands of celestial objects are observed every day by space missions such as Kepler, K2, and TESS. Manually analyzing these observations is time-consuming and challenging.

**A World Away** uses Artificial Intelligence to assist in the classification of potential exoplanets, helping researchers and astronomy enthusiasts quickly analyze observation data and determine whether an object is:

- ✅ Confirmed Exoplanet
- 🧩 Exoplanet Candidate
- ❌ False Positive

The project provides an interactive web interface built with Streamlit, allowing users to either enter astronomical parameters manually or upload datasets for bulk predictions.

---

## 🎯 Problem Statement

Modern space telescopes generate enormous amounts of observational data. Identifying exoplanets within this data requires significant computational effort and expert analysis.

Our goal is to:

- Reduce the time required for initial classification.
- Automate exoplanet candidate evaluation.
- Make astronomical data analysis more accessible.

---

## 💡 Solution

We developed a machine learning model trained on NASA exoplanet datasets and integrated it into a user-friendly Streamlit application.

The system:

1. Receives exoplanet observation parameters.
2. Processes the data through a trained neural network model.
3. Predicts the most likely classification.
4. Displays prediction confidence scores.
5. Supports both single-object and batch predictions.

---

## ✨ Features

### 🔮 Single Prediction Mode
Users can manually enter astronomical parameters and instantly receive a prediction.

### 📁 CSV Upload Support
Upload a CSV file containing multiple observations and classify them all at once.

### 📊 Confidence Scoring
Each prediction includes a confidence percentage generated by the model.

### 📥 Export Results
Prediction results can be downloaded as a CSV file.

### 🌐 Interactive Web Interface
Built with Streamlit for a simple and accessible user experience.

## 🧠 Model Inputs

The model uses the following astronomical features:

| Feature | Description |
|----------|------------|
| pl_orbper | Orbital Period |
| pl_tranmid | Transit Midpoint |
| pl_trandep | Transit Depth |
| pl_trandur | Transit Duration |
| pl_rade | Planet Radius |
| pl_eqt | Equilibrium Temperature |
| pl_insol | Insolation Flux |
| st_teff | Stellar Effective Temperature |
| st_rad | Stellar Radius |
| st_logg | Stellar Surface Gravity |
| pl_ratror | Planet-Star Radius Ratio |
| st_mass | Stellar Mass |

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/mrwanamr65/Nasa-challange-world-away-.git
cd Nasa-challange-world-away-
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Application

Start the Streamlit server:

```bash
streamlit run strm.py
```

The application will open in your browser automatically.

|---------|---------|

| 🧩 Candidate | Potential exoplanet requiring further analysis |
| ❌ False Positive | Observation unlikely to be an exoplanet |
